# Personal Portfolio

My first portfolio website, built about a year into learning web development.
It's a work in progress and a snapshot of where I started.

## About

I taught myself HTML, CSS, and JavaScript for roughly a year before putting
this together. The goal was simple: get something real online, mess with it,
and keep learning by doing.

## Heads Up

The links inside the project pages are placeholders and don't actually work
yet. The site is mostly there to show layout, styling, and the small bits
of JS I figured out along the way.

## What I Practiced

- Structuring pages with semantic HTML
- Styling layouts with Flexbox and Grid
- Writing vanilla JS functions
- Playing with responsive design
- Hosting on GitHub Pages

## Run It Locally

1. Clone the repo
2. Open `index.html` in your browser

That's it. No installs, no dependencies.
